# 🧠 Food2Mood AI

> 음식과 생활 데이터를 기반으로 다음날 컨디션을 예측하고  
> 개인 맞춤 식단을 추천하는 AI 기반 헬스케어 웹 애플리케이션

---

## 📌 프로젝트 소개

Food2Mood AI는 단순한 칼로리 기록 앱이 아닌,  
**영양 성분 + 생활 습관 + 컨디션 데이터를 종합 분석하여  
다음날 컨디션 점수를 예측하는 개인화 건강 플랫폼**입니다.

기존 서비스가 ‘기록’에 집중했다면,  
본 프로젝트는 **예측과 추천**에 초점을 맞추었습니다.

---

## 🎯 기획 의도

- 음식이 실제로 내 몸에 어떤 영향을 주는지 알고 싶다는 문제의식
- 사용자가 복잡한 건강 데이터를 직관적으로 이해할 수 있는 UI 설계
- AI 예측 결과를 신뢰할 수 있도록 데이터 시각화 강화

---

## 🧩 주요 기능

### 1️⃣ 컨디션 예측
- 영양 성분, 수면 시간, 운동량 등을 기반으로
- 다음날 컨디션 점수(0~100) 예측
- 혈당 급상승 위험도 분류

### 2️⃣ 개인화 추천 시스템
- 유사 사용자 기반 협업 필터링
- 예측 점수 기반 음식 추천
- 리스크 최소화 식단 제안

### 3️⃣ 데이터 시각화
- 최근 7일 컨디션 추이 (Line Chart)
- 혈당 위험도 게이지 차트
- 당 섭취 vs 피로도 상관관계 그래프
- AI 영향도 시각화

### 4️⃣ 다크모드 지원
- 시스템 테마 기반 자동 전환
- Tailwind 다크모드 클래스 기반 설계

## 🧠 AI 모델 구조

### 1️⃣ 회귀 모델
- 목표: 다음날 컨디션 점수 예측
- 모델 비교: Linear Regression / RandomForest / XGBoost
- 평가 지표: RMSE, MAE, R²

### 2️⃣ 분류 모델
- 목표: 혈당 급상승 위험도 분류
- 평가 지표: F1-score, ROC-AUC

### 3️⃣ 추천 시스템
- K-means 기반 유사 사용자 군집화
- Hybrid 추천 (예측 점수 + 협업 필터링)
